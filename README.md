# AI Interview Practice Platform

A comprehensive full-stack web application that simulates real interviews using AI technology. Practice interviews with voice interaction, webcam analysis, and receive detailed AI-generated feedback.

## Features

- 🤖 **AI-Powered Interviewer**: Natural conversation with adaptive questioning
- 🎥 **Real-time Voice & Video**: Practice with voice interaction and webcam analysis
- 📊 **Confidence Analysis**: AI analyzes facial expressions and voice for confidence metrics
- 📝 **Resume-Based Questions**: Upload your resume for personalized questions
- 📈 **Detailed Reports**: Comprehensive performance analysis with actionable insights
- 🎯 **Multiple Subjects**: JavaScript, Python, Java, Data Structures, and more
- 📱 **Responsive Design**: Works seamlessly on all devices

## Tech Stack

### Frontend
- HTML5, CSS3, JavaScript
- Modern UI with glassmorphism effects
- Chart.js for data visualization
- Web Speech API for voice interaction
- WebRTC for webcam access

### Backend
- Node.js with Express.js
- MongoDB with Mongoose
- JWT Authentication
- bcrypt for password hashing
- OpenAI/Gemini API integration
- PDF parsing for resume analysis
